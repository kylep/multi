Each page in this section is a study of one external tool: what it
does, how it works, and whether it earns its place in this project's
stack. The lens is practical, not promotional. Where a project's
own marketing makes claims that aren't independently verifiable, the
page either flags the claim or omits it.

Tool-research differs from neighbouring sections:

- [AI Tools](/wiki/ai-tools.html) — short stubs of tools the
  agent team currently uses, with usage notes and hooks
- [Custom Tools](/wiki/custom-tools.html) — tools, skills, and
  Docker images built in-house
- [Research](/wiki/research.html) — Deep Research report archives
  across multiple providers
