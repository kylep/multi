## AI Industry

- [Anthropic](https://www.anthropic.com/news/claude-fable-5-mythos-5) Claude Fable 5 launched publicly on June 9, the most capable model Anthropic has made generally available. It is state-of-the-art on nearly all tested benchmarks and is priced at $10/M input and $50/M output tokens — less than half the previous Mythos Preview rate. Pro, Max, Team, and Enterprise users get access at no extra cost through June 22.

## AI Tooling

- [Devin AI Blog](https://devin.ai/blog/windsurf-is-now-devin-desktop/) Windsurf became Devin Desktop on June 2 via an over-the-air update. The renamed IDE ships Devin Local (a Rust rewrite of Cascade, 30% more token efficient), an Agent Command Center for managing all agents in one place, and support for the Agent Client Protocol (ACP). Cascade reaches end-of-life on July 1.

- [GitHub Blog](https://github.blog/news-insights/company-news/github-copilot-individual-plans-introducing-flex-allotments-in-pro-and-pro-and-a-new-max-plan/) GitHub Copilot launched a new $100/month Max plan on June 1 with 20,000 credits (10,000 base + 10,000 flex) designed for sustained agentic workloads. Usage-based billing is now live across all paid tiers; code completions remain unlimited and do not consume credits.

## Security

## Security

- [The Hacker News](https://thehackernews.com/2026/06/google-sues-chinese-smishing-network.html) Google filed a lawsuit June 12 against a China-based phishing-as-a-service ring called "Outsider Enterprise" for using Gemini AI to generate phishing code and fake websites. The network created 9,000+ fraudulent sites, sent 2.5 million spam texts in a two-week window, and reached 100,000+ victims. It is the first time Google has sued a threat actor for weaponizing its own AI platform.
