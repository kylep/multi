## AI Industry

- [CNBC](https://www.cnbc.com/2026/03/26/anthropic-pentagon-dod-claude-court-ruling.html) Judge Rita Lin issued a preliminary injunction blocking the Pentagon's supply-chain risk designation against Anthropic, ruling the designation amounts to First Amendment retaliation. The order bars the Trump administration from enforcing the directive while Anthropic's second D.C. lawsuit over civilian contract exclusions proceeds separately.

- [Bloomberg](https://www.bloomberg.com/news/articles/2026-03-27/claude-ai-maker-anthropic-said-to-weigh-ipo-as-soon-as-october) Anthropic has opened early discussions with Goldman Sachs, JPMorgan, and Morgan Stanley about a potential IPO as soon as October 2026, racing OpenAI to go public. The company was last valued at approximately $380 billion following a $30 billion MGX-led round that closed in February.

- [Fortune](https://fortune.com/2026/03/26/anthropic-says-testing-mythos-powerful-new-ai-model-after-data-leak-reveals-its-existence-step-change-in-capabilities/) A misconfigured CMS at Anthropic exposed approximately 3,000 unpublished internal assets including a draft blog post naming a new model called "Claude Mythos," internally codenamed "Capybara." Anthropic confirmed the model is in early access testing and described its cybersecurity capabilities as posing unprecedented risks.

## AI Tooling

- [Cloudflare Blog](https://blog.cloudflare.com/dynamic-workers/) Cloudflare launched Dynamic Workers in open beta on March 24 for all paid Workers users. The system runs AI-generated code in V8 isolates that start in milliseconds, roughly 100 times faster than traditional containers, and Cloudflare reports converting an MCP server to its TypeScript code mode cuts token usage by 81%.

- [Bloomberg](https://www.bloomberg.com/news/articles/2026-03-26/google-gemini-adds-tool-to-make-it-easier-to-switch-from-chatgpt) Google added a chat history import tool to Gemini, letting free and paid users upload zipped conversation exports from ChatGPT and Claude. The feature is live across all Gemini plan tiers.

## Open Source

- [GitHub](https://github.com/topics/artificial-intelligence-projects) obra/superpowers reached 92,100 GitHub stars as of the week of March 25 — a Shell-based agentic skills framework that lets developers compose AI agent behavior without Python boilerplate. It sits atop GitHub trending as the skills ecosystem continues to expand following Anthropic's open skills standard.

- [GitHub Trending](https://www.shareuhack.com/en/posts/github-trending-weekly-2026-03-25) flash-moe runs the Qwen3.5-397B-A17B mixture-of-experts model locally on a MacBook Pro with 48GB RAM using pure C and Apple Metal, with no cloud dependency. The project generated significant traction on Hacker News this week among developers tracking local large-model inference.

## Security

- [CISA](https://www.cisa.gov/known-exploited-vulnerabilities-catalog) CVE-2026-3909 in Google Chromium's V8 engine allows a remote attacker to execute arbitrary code inside a sandbox via a crafted HTML page; the vulnerability affects Chrome, Edge, and Opera. CISA's remediation deadline for federal agencies was today, March 27.
