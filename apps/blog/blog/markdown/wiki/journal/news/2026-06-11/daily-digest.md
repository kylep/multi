## AI Industry

- [Anthropic Newsroom](https://www.anthropic.com/news) Anthropic confidentially submitted a draft S-1 to the SEC on June 1, joining OpenAI (filed June 8) in a wave of major AI IPO preparations. Goldman Sachs is reportedly tracking both offerings, with an AI IPO window expected between September and November 2026.

- [Microsoft News](https://news.microsoft.com/source/2026/06/09/kpmg-and-microsoft-scale-trusted-enterprise-ai-agents-globally-through-deployment-of-agent-365-and-copilot/) KPMG is deploying Microsoft 365 Copilot across its entire global workforce of over 276,000 professionals, the largest announced enterprise Copilot rollout to date. The deployment includes Agent 365, Microsoft's agentic workflow layer for M365 apps.

- [Google AI for Developers](https://ai.google.dev/gemini-api/docs/changelog) Google released Gemini Omni, a new multimodal model that generates any output from any input including video. Google is simultaneously deprecating the gemini-3.1-flash-image-preview and gemini-3-pro-image-preview models, with shutdown set for June 25.

## AI Tooling

- [Claude Code Changelog](https://code.claude.com/docs/en/changelog) Claude Code's June 9 release adds a --safe-mode flag (also settable via CLAUDE_CODE_SAFE_MODE) that starts a session with all customizations disabled for clean troubleshooting, plus a /cd command to change working directory without breaking the prompt cache mid-session. The update also improves remote session handling, background agent reliability, and enterprise policy enforcement.

- [Microsoft Learn](https://learn.microsoft.com/en-us/microsoft-365/copilot/release-notes) Microsoft IQ is now generally available across GitHub Copilot, Foundry, and Copilot Studios. The context layer combines Work IQ from M365 signals, Fabric IQ for structured business data, and a new Web IQ grounding layer to feed agents real workplace knowledge.

## Security

- [Zero Day Initiative](https://www.zerodayinitiative.com/blog/2026/6/9/the-june-2026-security-update-review) The June 2026 Patch Tuesday included a critical HTTP.sys Remote Code Execution flaw (CVE-2026-47291) allowing unauthenticated remote attackers to execute code on affected systems with no user interaction. This was among 32 critical CVEs patched in the record-breaking 198-CVE June release.

- [The Hacker News](https://thehackernews.com/) CVE-2026-3300, a critical PHP eval() injection vulnerability in Everest Forms Pro through version 1.9.12, is being actively exploited in the wild for unauthenticated remote code execution. Patch to a version beyond 1.9.12 to remediate.
