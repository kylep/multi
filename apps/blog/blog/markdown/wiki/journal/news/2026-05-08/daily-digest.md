## Security

- [The Hacker News](https://thehackernews.com/2026/05/cisa-adds-actively-exploited-linux-root.html) CISA added CVE-2026-31431 (dubbed "Copy Fail") to its Known Exploited Vulnerabilities catalog. The Linux kernel local privilege escalation flaw allows an unprivileged user to obtain root access and affects Red Hat, SUSE, Ubuntu, and AWS Linux distributions.

- [The Hacker News](https://thehackernews.com/2026/05/palo-alto-pan-os-flaw-under-active.html) CISA added CVE-2026-0300 to the KEV catalog on May 6. The Palo Alto PAN-OS flaw carries a CVSS score of 9.3 and allows unauthenticated remote code execution when the User-ID Authentication Portal is internet-exposed.

- [DailyCVE](https://dailycve.com/argo-cd-authorization-bypass-cve-2026-43824-critical/) CVE-2026-43824, a critical authorization bypass in Argo CD, was disclosed May 7. The `ServerSideDiff` endpoint exposes Kubernetes Secret values to any authenticated user by default. Patched versions are 3.3.9 and 3.2.11.

## Security

- [Krebs on Security](https://krebsonsecurity.com/2026/05/canvas-breach-disrupts-schools-colleges-nationwide/) ShinyHunters claimed a breach of Instructure's Canvas learning platform affecting 275 million students and faculty at nearly 9,000 schools. Stolen data includes names, email addresses, student IDs, and private messages. Canvas was taken offline briefly during finals week before being restored; ShinyHunters set a May 12 deadline for negotiations.

- [Bleeping Computer](https://www.bleepingcomputer.com/news/security/new-linux-dirty-frag-zero-day-with-poc-exploit-gives-root-privileges/) "Dirty Frag" (CVE-2026-43284, CVE-2026-43500) is a new Linux kernel privilege escalation pairing two bugs in the IPsec ESP and rxrpc fast-path decryption code. A working exploit yields root from any unprivileged user on Ubuntu, RHEL, openSUSE, AlmaLinux, and Fedora. A partial patch for the ESP component landed in the upstream netdev tree May 7; the rxrpc fix is still pending.
