## AI Industry

- [NBC News](https://www.nbcnews.com/business/corporations/anthropic-files-ipo-openai-rcna347897) Anthropic filed confidentially for an IPO on June 1, becoming the first major AI lab to initiate the process. OpenAI is expected to file within weeks.

- [Crypto Briefing](https://cryptobriefing.com/us-government-equity-stakes-openai-anthropic/) President Trump confirmed the White House is examining the possibility of government equity stakes in OpenAI to ensure ordinary Americans benefit from AI's advance. Anthropic publicly confirmed it is not part of those discussions.

- [NVIDIA Newsroom](https://nvidianews.nvidia.com/news/meta-builds-ai-infrastructure-with-nvidia) NVIDIA and Meta announced a multiyear, multigenerational strategic partnership covering on-premises and cloud AI infrastructure, including deployment of millions of Blackwell and Rubin GPUs and NVIDIA Spectrum-X networking.

- [TechCrunch](https://techcrunch.com/2026/06/01/nvidia-chases-200b-cpu-market-with-ai-agent-pcs-from-microsoft-dell-and-hp/) NVIDIA unveiled the RTX Spark "superchip" at Computex, entering the PC CPU market. AI PCs from ASUS, Dell, HP, Lenovo, Microsoft Surface, and MSI are slated for this fall.

- [Tom's Guide](https://www.tomsguide.com/news/live/microsoft-build-2026) Microsoft Build 2026 (June 2–3) centered on agentic AI: seven new MAI models were introduced, including MAI-Thinking-1 (35B parameters, 256K context, built without distillation) and the long-running autonomous agent framework called Autopilots.

## AI Tooling

- [HiTechies](https://www.hitechies.com/github-copilot-metered-billing-june-2026/) GitHub Copilot moved to token-based AI Credits billing on June 1. The $10 Pro plan now includes 1,500 credits; a new Copilot Max tier at $100/month offers 20,000 credits. Premium request units are retired.

- [Releasebot](https://releasebot.io/updates/anthropic) Anthropic expanded Project Glasswing, extending Claude Mythos Preview access to approximately 150 new organizations. Claude Security for codebase scanning and patch suggestions was also added. A bug fix for Claude Code addresses an Opus 4.8 issue where modified thinking blocks caused API errors.

- [Space Daily](https://spacedaily.com/n-microsoft-is-canceling-claude-code-licenses-across-its-experiences-devices-division-by-june-30-steering-thousands-of-engineers-toward-github-copilot-while-uber-burned-through-its-entire-2026-ai-bu/) Microsoft is cancelling Claude Code licences across its Experiences and Devices division by June 30 and redirecting engineers to GitHub Copilot CLI. Separately, Uber exhausted its entire 2026 AI tools budget on Claude Code and Cursor in four months.

- [Tom's Guide](https://www.tomsguide.com/news/live/microsoft-build-2026) MAI-Code-1-Flash began rolling out in VS Code via the GitHub Copilot model picker, tuned for fast, low-cost coding. The GitHub Copilot desktop app launched in preview with agentic workflow support.

## Open Source

## Security

- [Help Net Security](https://www.helpnetsecurity.com/2026/06/05/cisco-sd-wan-cve-2026-20245-0-day-exploited/) Cisco disclosed a zero-day in Catalyst SD-WAN Manager (CVE-2026-20245) with no patch currently available. Mandiant reported the flaw; Cisco confirmed limited active exploitation that resulted in configuration changes pushed to edge devices. All deployment types are affected, including FedRAMP government.

- [The Hacker News](https://thehackernews.com/2026/05/microsoft-warns-of-two-actively.html) CISA added two actively exploited Microsoft Defender vulnerabilities (CVE-2026-41091 and CVE-2026-45498) to its Known Exploited Vulnerabilities catalog, with a remediation deadline of June 3 for federal agencies.

- [The Cyber Express](https://thecyberexpress.com/cve-2025-48595-android-june-2026/) Google's June 2026 Android security update patches 124 vulnerabilities, including one actively exploited zero-day — CVE-2025-48595, an integer overflow in the Android Framework that enables local privilege escalation without user interaction.
