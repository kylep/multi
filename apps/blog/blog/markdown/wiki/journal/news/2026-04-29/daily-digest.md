## AI Industry

- [Tom's Hardware](https://www.tomshardware.com/tech-industry/artificial-intelligence/anthropic-surpasses-biggest-rival-openai-in-secondary-market-valuation-surges-to-usd1-trillion-amid-frantic-investor-interest) Anthropic's secondary market valuation reached $1 trillion, overtaking OpenAI's $852B post-money figure from its March fundraise. Anthropic's annualized revenue run-rate tripled to $30B by end of March from $9B in late 2025.

Analysts expect EPS of $4.04 on revenue of $81.46B, with attention on Azure AI growth following the company's restructured OpenAI partnership.

- [Kubernetes Blog](https://kubernetes.io/blog/2026/01/29/ingress-nginx-statement/) Ingress-NGINX was retired in early 2026 — no further releases, security patches, or bug fixes will be issued. Kubernetes recommends migration to Gateway API; F5 maintains an Apache 2.0-licensed NGINX Ingress Controller as a drop-in alternative.

## Security

- [The Hacker News](https://thehackernews.com/2026/04/docker-cve-2026-34040-lets-attackers.html) CVE-2026-34040 (CVSS 8.8) in Docker Engine is an incomplete fix for CVE-2024-41110 that lets attackers bypass AuthZ plugin inspection by padding API requests beyond 1MB. The daemon processes the full oversized request, enabling creation of privileged containers with root host access. Fix is in Docker Engine 29.3.1.

## Security

- [The Hacker News](https://thehackernews.com/2026/04/researchers-discover-critical-github.html) CVE-2026-3854 (CVSS 8.7) is a command injection flaw in GitHub's git push pipeline that allows any authenticated user to execute arbitrary code on GitHub's backend servers with a single push. Wiz discovered and reported it March 4; GitHub deployed a patch to GitHub.com within two hours. GitHub Enterprise Server versions 3.14.25 and later are patched. No evidence of exploitation in the wild.
