## Security

- [CNN](https://www.cnn.com/2026/05/07/us/canvas-hack-strands-college-students-finals-week) ShinyHunters breached Instructure's Canvas LMS and claimed to have stolen 3.65 TB of data affecting approximately 275 million students and faculty across ~9,000 schools worldwide. Data reportedly included names, email addresses, student IDs, and private messages. Instructure says it reached an agreement with the attackers and the data was destroyed.

- [SecurityWeek](https://www.securityweek.com/new-dirty-frag-linux-vulnerability-possibly-exploited-in-attacks/) CVE-2026-42945 (NGINX Rift) is a heap buffer overflow in nginx's rewrite module affecting all nginx builds since 2008, allowing unauthenticated remote code execution. Cloudflare deployed emergency detection rules. Two additional Linux privilege escalation flaws, Dirty Frag and Copy Fail 2, allow unprivileged users to reach root.

- [The Hacker News](https://thehackernews.com/2026/05/on-prem-microsoft-exchange-server-cve.html) CVE-2026-42897 is an XSS vulnerability in on-premises Microsoft Exchange Server that lets attackers execute arbitrary JavaScript in Outlook Web Access via a crafted email. A patch is available.

## Security

- [The Hacker News](https://thehackernews.com/2026/05/laravel-lang-php-packages-compromised.html) A supply chain attack compromised 233 versions of Laravel-Lang packages across 700 GitHub repositories, detected May 22–23. The attacker rewrote historical version tags in the laravel-lang/lang, laravel-lang/http-statuses, laravel-lang/attributes, and laravel-lang/actions packages to point to malicious forks containing a 5,900-line PHP credential stealer that harvests AWS, GCP, Azure, Kubernetes, SSH, and browser credentials. Packagist took down the affected versions and temporarily unlisted the packages.
