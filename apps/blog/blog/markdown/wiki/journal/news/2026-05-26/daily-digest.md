## Security

- [Security Boulevard](https://securityboulevard.com/2026/05/three-cves-and-the-may-2026-exploit-chain-nobodys-taking-seriously/) CVE-2026-42945 (NGINX Rift) is a heap buffer overflow in ngx_http_rewrite_module present since 2008, allowing unauthenticated remote code execution via a crafted HTTP request. It is part of a three-CVE Linux exploit chain including Copy Fail (CVE-2026-31431) and Dirty Frag (CVE-2026-43284/43500), both of which enable unprivileged local escalation to root.

- [Help Net Security](https://www.helpnetsecurity.com/2026/05/21/microsoft-defender-vulnerabilities-cve-2026-41091-cve-2026-45498/) Two Microsoft Defender vulnerabilities — CVE-2026-41091 and CVE-2026-45498 — are being actively exploited in the wild. Microsoft has released patches and CISA has added both to its Known Exploited Vulnerabilities catalog.

- [Security reporting](https://securityboulevard.com/2026/05/three-cves-and-the-may-2026-exploit-chain-nobodys-taking-seriously/) 7-Eleven confirmed a breach involving over 600,000 Salesforce records following a ransom demand. Compromised data includes customer and internal records. The company is notifying affected individuals.

## Security

- [The Hacker News](https://thehackernews.com/2026/05/microsoft-releases-mitigation-for.html) CVE-2026-45585 (YellowKey) is a zero-day BitLocker bypass that requires physical access — an attacker boots into Windows Recovery Environment from a crafted USB or EFI partition to unlock an encrypted drive without the PIN. Affects Windows 11 24H2 through 26H1 and Windows Server 2025. Microsoft released a mitigation on May 20; switching BitLocker from TPM-only to TPM+PIN mode blocks exploitation.
