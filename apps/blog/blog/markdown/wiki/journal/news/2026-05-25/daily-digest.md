## AI Industry

- [TechCrunch](https://techcrunch.com/2026/05/19/openai-co-founder-andrej-karpathy-joins-anthropics-pre-training-team/) Andrej Karpathy, OpenAI co-founder and former Tesla Autopilot lead, joined Anthropic's pre-training team on May 19. He will help build a new team focused on using Claude itself to accelerate pre-training research under team lead Nick Joseph.

- [Anthropic](https://www.anthropic.com/news/gates-foundation-partnership) Anthropic announced a four-year partnership with the Gates Foundation committing $200 million in grants, Claude usage credits, and technical support for programs in global health, life sciences, education, and economic mobility.

- [Google Blog](https://blog.google/innovation-and-ai/sundar-pichai-io-2026/) Google I/O 2026 introduced Gemini 3.5 Flash (4x faster than prior frontier models), Gemini Omni (multimodal video simulation combining Gemini reasoning with DeepMind's Veo and Genie), and Gemini Spark (a general-purpose agent that reasons across connected apps). Gemini 3.5 Flash becomes the default model for the Gemini app globally.

## AI Tooling

- [Google Blog](https://blog.google/innovation-and-ai/technology/developers-tools/google-io-2026-developer-highlights/) Google launched Antigravity 2.0 at I/O 2026, a standalone agent-first coding platform that orchestrates multiple autonomous agents in parallel. A demo built a functioning operating system from scratch in 12 hours using 93 subagents at under $1,000 in API credits.

- [GitHub Changelog](https://github.blog/changelog/2026-05-20-updates-to-available-models-in-copilot-on-web/) GitHub Copilot released a desktop app technical preview for agentic coding workflows, with an inbox-style interface surfacing issues, PRs, and CI checks across repositories. Copilot on web also added semantic issue search and removed all Gemini models, retaining OpenAI and Claude models.

- [Anthropic](https://www.anthropic.com/news/gates-foundation-partnership) Anthropic expanded Project Glasswing with Claude Security entering public beta, adding new cyber verification tools for eligible security teams.

## Security

- [CNN](https://www.cnn.com/2026/05/07/us/canvas-hack-strands-college-students-finals-week) ShinyHunters breached Instructure's Canvas learning platform in late April, claiming 3.65 TB of data from roughly 275 million users across 8,809 educational institutions. Compromised data includes names, email addresses, student IDs, and internal messages. Instructure states passwords and financial data were not involved and later claimed the stolen data was destroyed following an agreement with the attackers.

- [The Hacker News](https://thehackernews.com/2026/05/on-prem-microsoft-exchange-server-cve.html) CVE-2026-42897 is a spoofing vulnerability in on-premises Microsoft Exchange Server. An attacker can trigger arbitrary JavaScript execution in Outlook Web Access by sending a crafted email; a patch is available.

- [SecurityWeek](https://www.securityweek.com/cisco-patches-another-sd-wan-zero-day-the-sixth-exploited-in-2026/) Cisco patched CVE-2026-20182, an authentication bypass in SD-WAN that grants remote admin access via crafted packets. This is the sixth Cisco SD-WAN zero-day actively exploited in 2026, attributed to threat actor UAT-8616.
