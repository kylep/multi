## AI Tooling

## Security

- [The Hacker News](https://thehackernews.com/search/label/Vulnerability) FortiClient EMS CVE-2026-35616 (CVSS 9.1) — a pre-authentication API bypass that leads to privilege escalation — is under active exploitation. Organizations running FortiClient EMS should patch immediately.

- [Threat Modeling](https://threat-modeling.com/vulnerability-intelligence-report-may-25-2026/) TanStack supply chain attack: malicious versions of TanStack packages were published to npm and used to distribute credential-stealing malware. Developers should verify installed versions.

- [NVD](https://nvd.nist.gov/vuln/detail/CVE-2026-46333) Linux kernel CVE-2026-46333 went undetected for nine years — an unprivileged local user can disclose sensitive files and execute arbitrary commands as root. Patch is available in current kernel releases.
