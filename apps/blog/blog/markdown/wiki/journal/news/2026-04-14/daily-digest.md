## AI Tooling

- [GitHub](https://github.com/openclaw/openclaw/releases/tag/v2026.4.2) OpenClaw v2026.4.2 ships Durable Task Flow orchestration — background task state is now tracked, inspectable, and recoverable instead of silent. Release also hardens provider transport, tightens plugin activation boundaries, strips Anthropic thinking-block tags from user-visible output, and fixes tool-payload normalization for Kimi Coding compatibility.

- [TechCrunch](https://techcrunch.com/2026/04/13/microsoft-is-working-on-yet-another-openclaw-like-agent/) Microsoft is building a separate always-on Copilot agent modeled on OpenClaw, distinct from the OpenClaw integration announced April 10. The project, led by CVP Omar Shahine, targets enterprise customers with tighter security controls and 24/7 autonomous operation across M365. Expected to appear at Build in June.

## Security

- [The Hacker News](https://thehackernews.com/2026/04/new-chrome-zero-day-cve-2026-5281-under.html) Chrome zero-day CVE-2026-5281: use-after-free in Dawn (Google's WebGPU implementation) allows RCE via a crafted HTML page. Actively exploited in the wild — Google's fourth Chrome zero-day patched in 2026. CISA added it to the Known Exploited Vulnerabilities catalog; federal agencies must patch by April 15. Update to Chrome 146.0.7680.177/178 (also affects Edge, Brave, Opera, Vivaldi).

## AI Tooling

## Update — 19:00 UTC

## AI Industry

- [Axios](https://www.axios.com/2026/04/13/anthropic-revenue-growth-ai) Anthropic's annualized revenue reached $30B in April 2026, surpassing OpenAI's reported $25B ARR. Growth from $9B at end of 2025 to $30B now represents roughly 30x expansion in 15 months. OpenAI disputed the figure, calling Anthropic's accounting methodology inflated.

- [Anthropic](https://www.anthropic.com/glasswing) Anthropic disclosed Project Glasswing — a responsible-disclosure program triggered by capabilities observed in the Claude Mythos2 Preview model. The model demonstrated an ability to find and exploit software vulnerabilities surpassing all but the most skilled human researchers; Anthropic is withholding general release pending safety review.

- [DNyuz](https://dnyuz.com/2026/04/14/in-the-wake-of-anthropics-mythos-openai-has-a-new-cybersecurity-model-and-strategy/) OpenAI announced GPT-5.4-Cyber, a model purpose-built for defensive security use cases. The announcement comes one week after Anthropic disclosed Claude Mythos's offensive security capabilities.

- [OneNewsPage](https://www.onenewspage.com/n/Internet/1ztf28lom1/Meta-commits-another-21-billion-to-CoreWeave-bringing.htm) Meta committed an additional $21B to CoreWeave for dedicated AI cloud capacity running 2027 through 2032, bringing the total relationship to $35B. New contract covers early deployments of NVIDIA's Vera Rubin platform across multiple sites.
