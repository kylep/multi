## AI Tooling

- [GitHub](https://github.com/openclaw/openclaw/releases/tag/v2026.4.2) OpenClaw v2026.4.2 ships Durable Task Flow orchestration — background task state is now tracked, inspectable, and recoverable instead of silent. Release also hardens provider transport, tightens plugin activation boundaries, strips Anthropic thinking-block tags from user-visible output, and fixes tool-payload normalization for Kimi Coding compatibility.

- [TechCrunch](https://techcrunch.com/2026/04/13/microsoft-is-working-on-yet-another-openclaw-like-agent/) Microsoft is building a separate always-on Copilot agent modeled on OpenClaw, distinct from the OpenClaw integration announced April 10. The project, led by CVP Omar Shahine, targets enterprise customers with tighter security controls and 24/7 autonomous operation across M365. Expected to appear at Build in June.

## Security

- [The Hacker News](https://thehackernews.com/2026/04/new-chrome-zero-day-cve-2026-5281-under.html) Chrome zero-day CVE-2026-5281: use-after-free in Dawn (Google's WebGPU implementation) allows RCE via a crafted HTML page. Actively exploited in the wild — Google's fourth Chrome zero-day patched in 2026. CISA added it to the Known Exploited Vulnerabilities catalog; federal agencies must patch by April 15. Update to Chrome 146.0.7680.177/178 (also affects Edge, Brave, Opera, Vivaldi).
