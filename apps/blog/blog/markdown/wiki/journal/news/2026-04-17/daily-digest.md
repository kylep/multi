## AI Industry

- [CNBC](https://www.cnbc.com/amp/2026/04/16/anthropic-london-office-800-staff-openai-expansion.html) Anthropic is expanding its London office to accommodate 800 staff, up from the current 200, in the Knowledge Quarter district. The move follows both OpenAI opening a permanent London hub and Anthropic's rift with the Pentagon over AI military use.

## AI Tooling

- [Releasebot](https://releasebot.io/updates/anthropic/claude-code) Claude Code v2.1.110 (April 16): adds `/tui fullscreen` for flicker-free rendering, mobile push notifications when Remote Control is enabled, a separate `/focus` command replacing the old `Ctrl+O` focus toggle, improved plugin interface, and MCP reliability fixes for dropped server connections. Also ships `/ultrareview` (parallel cloud multi-agent code review), `/less-permission-prompts` (auto-generates allowlists from transcript history), voice STT in 10 new languages (20 total), and an interactive effort slider.

- [The Tech Portal](https://thetechportal.com/2026/04/17/openai-upgrades-codex-with-multi-agent-workflows-and-desktop-app-control-to-challenge-anthropics-claude-code) OpenAI released a major Codex update: multi-agent workflows run simultaneously across writing, debugging, and testing tasks; macOS desktop app control lets agents interact with other applications; a built-in browser adds inline commenting for web projects; image generation uses gpt-image-1.5; memory persists context across sessions; and 111 new plugins ship on day one.

## Open Source

## Security

- [The Hacker News](https://thehackernews.com/2026/04/operation-poweroff-seizes-53-ddos.html) Operation PowerOFF: a 21-country law enforcement action (Europol-backed) seized 53 DDoS-for-hire domains, arrested four operators, obtained data on over 3 million criminal accounts, and sent warnings to 75,000 identified users. Law enforcement is now targeting search engine placement to disrupt recruitment of new users.

## Geopolitics
