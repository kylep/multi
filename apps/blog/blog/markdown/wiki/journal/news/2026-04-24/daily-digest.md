## Security

- [The Hacker News](https://thehackernews.com/2026/04/sglang-cve-2026-5760-cvss-98-enables.html) CVE-2026-5760 (CVSS 9.8) is a critical remote code execution flaw in SGLang, a widely used open-source LLM serving framework. Attackers can trigger arbitrary code execution by distributing a malicious GGUF model file containing an unsandboxed Jinja2 chat template; the attack surface includes Hugging Face model hosting.

- [Cybersecurity News](https://cybersecuritynews.com/microsoft-patch-tuesday-april-2026/) Microsoft's April 2026 Patch Tuesday addressed 168 vulnerabilities, including one actively exploited zero-day (CVE-2026-32201, a SharePoint Server spoofing vulnerability) and one publicly disclosed flaw.

- [The Hacker News](https://thehackernews.com/2026/04/fortinet-patches-actively-exploited-cve.html) Fortinet patched CVE-2026-35616 (CVSS 9.1) in FortiClient EMS, a pre-authentication API bypass enabling privilege escalation that has been actively exploited since March 31. CISA added it to the Known Exploited Vulnerabilities catalog on April 6.

## Security

- [The Hacker News](https://thehackernews.com/2026/04/adobe-patches-actively-exploited.html) CVE-2026-34621 (CVSS 8.6) is a prototype pollution flaw in Adobe Acrobat Reader enabling arbitrary code execution when a user opens a malicious PDF. Adobe patched it on April 13 after confirming active exploitation; CISA added it to the KEV catalog the same day. The flaw had been exploited undetected since at least December 2025.
