## Security

- [CNN](https://www.cnn.com/2026/05/07/us/canvas-hack-strands-college-students-finals-week) ShinyHunters breached Instructure's Canvas LMS in late April, claiming 3.65 TB of data from approximately 275 million users at roughly 9,000 schools. Instructure paid the ransom on May 11 and received a claim that the data was destroyed. No evidence of passwords, dates of birth, or financial data being taken; names, email addresses, and student IDs were confirmed exposed.

## Update — 16:04 UTC

## AI Industry

- [TechCrunch](https://techcrunch.com/2026/05/20/openai-claims-it-solved-an-80-year-old-math-problem-for-real-this-time/) OpenAI announced a general-purpose reasoning model independently disproved the Erdős planar unit distance conjecture — an open problem in discrete geometry since 1946. The model discovered constructions using algebraic number theory that beat square grids for maximizing unit-distance pairs among n points. Multiple independent mathematicians verified the proof. It is the first time AI autonomously resolved a prominent open math problem without being trained specifically for it.

## AI Tooling

- [Cloudflare Blog](https://blog.cloudflare.com/browser-run-containers/) Cloudflare rebuilt Browser Run on its own Containers platform, raising simultaneous browser capacity from 30 to 120 (4x) and cutting response times by 50%. The rebuild also adds WebGL and WebMCP support. Existing users required no changes. The update completes Cloudflare's six-layer agent infrastructure stack alongside Dynamic Workers, Workflows, Agent Memory, and Stripe Projects.

## Security

- [The Hacker News](https://thehackernews.com/2026/05/cisco-patches-cvss-100-secure-workload.html) Cisco patched CVE-2026-20223 (CVSS 10.0) in Cisco Secure Workload, a REST API authentication bypass allowing unauthenticated remote attackers to cross tenant boundaries and gain Site Admin privileges over data center and cloud configuration. No workaround exists; upgrade to 3.10.8.3 or 4.0.3.17. No active exploitation reported yet.

- [The Hacker News](https://thehackernews.com/2026/05/cisa-adds-exploited-langflow-and-trend.html) CISA added CVE-2025-34291 (CVSS 9.4) in Langflow to its KEV catalog on May 21. The origin validation error chains permissive CORS, missing CSRF protection, and a code-execution endpoint; exploits expose all stored API keys and cascade into downstream SaaS services. Iranian state group MuddyWater is actively exploiting it. FCEB remediation deadline: June 4.
