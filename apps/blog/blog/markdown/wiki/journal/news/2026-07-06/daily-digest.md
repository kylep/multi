## Security

- [The Hacker News](https://thehackernews.com/2026/07/new-bad-epoll-linux-kernel-flaw-lets.html) CVE-2026-46242 "Bad Epoll" is a Linux kernel use-after-free flaw that lets any unprivileged local user escalate to root. It affects kernel versions 5.10 through 6.11, a public 99%-reliable exploit exists, and many distributions have not yet shipped backport patches. No workaround is available as epoll cannot be disabled.
