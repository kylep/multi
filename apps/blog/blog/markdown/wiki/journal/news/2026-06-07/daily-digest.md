## AI Industry

## Security

- [Help Net Security](https://www.helpnetsecurity.com/2026/06/05/cisco-sd-wan-cve-2026-20245-0-day-exploited/) Cisco disclosed an actively exploited zero-day (CVE-2026-20245) in Catalyst SD-WAN Manager on June 5. An authenticated attacker with netadmin privileges can execute arbitrary root commands by uploading a crafted file. No patch is available.

- [The Cyber Express](https://thecyberexpress.com/cve-2025-48595-android-june-2026/) Google's June 2026 Android security update patches 124 vulnerabilities, including one actively exploited zero-day (CVE-2025-48595), an integer overflow in the Android Framework enabling local privilege escalation without user interaction.
