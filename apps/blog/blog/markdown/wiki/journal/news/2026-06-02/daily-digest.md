## AI Industry

- [Windows Forum](https://windowsforum.com/threads/microsoft-build-2026-mai-image-2-5-mai-voice-2-and-mai-transcribe-1-5.420924/) Microsoft Build 2026 opened today in San Francisco with the debut of the MAI model family: MAI-Thinking-1 (Microsoft's first dedicated reasoning model, trained without distillation), MAI-Image-2.5, MAI-Voice-2, and MAI-Transcribe-1.5. The models are targeting Copilot, Teams, Azure Speech, and Microsoft Foundry — marking the first time Microsoft's core developer products will run on Microsoft-built models rather than OpenAI's.

- [Windows News](https://windowsnews.ai/article/microsoft-build-2026-homegrown-ai-models-to-power-github-copilot.420887) Project Polaris, Microsoft's homegrown coding model unveiled at Build 2026, will replace GPT-4 Turbo as the default GitHub Copilot model starting August 2026. Polaris uses a mixture-of-experts architecture with chain-of-thought and tree-of-thought reasoning at inference time, outperforms GPT-4 Turbo on HumanEval and MBPP, and runs on custom Maia AI accelerators in Azure. Pro subscribers get multi-file context up to 100,000 lines and autonomous test generation. A three-month opt-out window allows teams to stay on GPT-4.

- [NVIDIA Newsroom](https://nvidianews.nvidia.com/news/nvidia-microsoft-windows-pcs-agents-rtx-spark) NVIDIA unveiled the RTX Spark superchip at Computex 2026: a 20-core ARM Grace CPU paired with a Blackwell GPU (6,144 CUDA cores, fifth-generation Tensor Cores) connected via NVLink-C2C, with up to 128GB LPDDR5X unified memory, 300 GB/s memory bandwidth, and 1 petaflop of AI performance. The chip supports 120B-parameter LLMs locally with 1M-token context. Laptops and compact desktops from ASUS, Dell, HP, Lenovo, Microsoft Surface, and MSI arrive fall 2026.
