## AI Tooling

- [Developer Tech](https://www.developer-tech.com/news/microsoft-claude-code-github-copilot-cli/) Microsoft is canceling Claude Code licenses across its Experiences+Devices division — the teams behind Windows, Microsoft 365, Teams, and Surface — by June 30, steering engineers to GitHub Copilot CLI instead. ## Security

- [BleepingComputer](https://www.bleepingcomputer.com/news/security/drupal-critical-sql-injection-flaw-now-targeted-in-attacks/) CVE-2026-9082, a critical SQL injection in Drupal core (versions 8.9.0–11.3.9), is actively exploited. Attackers hit nearly 6,000 sites across 65 countries within 48 hours of disclosure. The flaw affects PostgreSQL-backed installations and enables full data exposure, privilege escalation, and in some cases remote code execution. Fixed in versions 11.3.10, 11.2.12, 10.6.9, 10.5.10, and 10.4.10.

- [Help Net Security](https://www.helpnetsecurity.com/2026/05/21/microsoft-defender-vulnerabilities-cve-2026-41091-cve-2026-45498/) Two Microsoft Defender vulnerabilities are actively exploited: CVE-2026-41091 (CVSS 7.8, local privilege escalation to SYSTEM) and CVE-2026-45498 (denial of service). CISA requires federal civilian agencies to apply patches by June 3.
