## AI Tooling

- [The Register](https://www.theregister.com/ai-and-ml/2026/06/03/no-longer-just-a-copilot-microsofts-ai-wants-to-take-the-wheel/) Microsoft Build 2026 introduced Scout, the first entry in a new "Autopilot" category of agents that run continuously in the background without being prompted. Scout monitors apps, calendars, email, and documents and can take action autonomously across Microsoft 365.

- [Engadget](https://www.engadget.com/2185601/microsoft-build-2026-live-blog-copilot-windows-news/) Microsoft announced seven MAI models at Build 2026. MAI-Thinking-1, a 35B-parameter reasoning model with a 256K context window, entered private preview in Azure Foundry. MAI-Image-2.5 and its Flash variant are already live inside PowerPoint and OneDrive.

- [Cloudflare Blog](https://releasebot.io/updates/cloudflare) Cloudflare released self-managed OAuth on June 4, letting developers create and manage their own OAuth applications with scoped consent and domain verification for Cloudflare integrations. A June 3 update also lets developers attach cron schedules directly to Workflow bindings without separate Workers.

## Security

- [Threat-Modeling.com](https://threat-modeling.com/vulnerability-intelligence-report-june-2-2026/) Fortinet is reporting large-scale exploitation of Citrix NetScaler CVE-2026-3055 (CVSS 9.8), a critical authentication bypass in NetScaler ADC and Gateway. Organizations running on-premises NetScaler appliances should patch immediately.

- [The Hacker News](https://thehackernews.com/2026/05/microsoft-warns-of-two-actively.html) Belgium's government issued an urgent warning about active exploitation of CVE-2026-41089 (CVSS 9.8), a critical Windows Netlogon vulnerability. Attackers can use it to authenticate as any domain account without knowing credentials.

The report maps findings against existing MITRE ATT&CK and ATLAS frameworks.
