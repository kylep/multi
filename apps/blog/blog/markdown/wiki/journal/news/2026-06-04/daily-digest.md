## AI Tooling

- [The Register](https://www.theregister.com/ai-and-ml/2026/06/03/no-longer-just-a-copilot-microsofts-ai-wants-to-take-the-wheel/) Microsoft Build 2026 introduced Scout, the first entry in a new "Autopilot" category of agents that run continuously in the background without being prompted. Scout monitors apps, calendars, email, and documents and can take action autonomously across Microsoft 365.

- [Engadget](https://www.engadget.com/2185601/microsoft-build-2026-live-blog-copilot-windows-news/) Microsoft announced seven MAI models at Build 2026. MAI-Thinking-1, a 35B-parameter reasoning model with a 256K context window, entered private preview in Azure Foundry. MAI-Image-2.5 and its Flash variant are already live inside PowerPoint and OneDrive.

- [Cloudflare Blog](https://releasebot.io/updates/cloudflare) Cloudflare released self-managed OAuth on June 4, letting developers create and manage their own OAuth applications with scoped consent and domain verification for Cloudflare integrations. A June 3 update also lets developers attach cron schedules directly to Workflow bindings without separate Workers.

## Security

- [Threat-Modeling.com](https://threat-modeling.com/vulnerability-intelligence-report-june-2-2026/) Fortinet is reporting large-scale exploitation of Citrix NetScaler CVE-2026-3055 (CVSS 9.8), a critical authentication bypass in NetScaler ADC and Gateway. Organizations running on-premises NetScaler appliances should patch immediately.

- [The Hacker News](https://thehackernews.com/2026/05/microsoft-warns-of-two-actively.html) Belgium's government issued an urgent warning about active exploitation of CVE-2026-41089 (CVSS 9.8), a critical Windows Netlogon vulnerability. Attackers can use it to authenticate as any domain account without knowing credentials.

## Update — 18:00 UTC

## AI Industry

- [Fortune](https://fortune.com/2026/06/01/anthropic-confidentially-files-ipo-965-billion-valuation/) Anthropic filed a confidential S-1 with the SEC on June 1, following a $65 billion Series H that set its valuation at $965 billion. Revenue run-rate reached roughly $47 billion in May 2026, up from $10 billion the prior year. An October 2026 listing window is widely expected.

- [Google Blog](https://blog.google/innovation-and-ai/technology/ai/google-io-2026-all-our-announcements/) Google launched Gemini 3.5 Flash, the first model in a series combining what Google calls "frontier intelligence with action." It outperforms Gemini 3.1 Pro on coding and agentic benchmarks while keeping Flash-tier latency, and is rolling out to all Google AI Plus, Pro, and Ultra subscribers.

## AI Tooling

- [Devin Blog](https://devin.ai/blog/windsurf-is-now-devin-desktop/) Cognition renamed Windsurf to Devin Desktop on June 2. The update — delivered as an over-the-air push — replaces the Cascade engine with Devin Local (rewritten in Rust, 30% more token-efficient), adds an Agent Command Center Kanban dashboard for managing local and cloud agents, and ships support for the open Agent Client Protocol, allowing Codex, Claude Agent, and OpenCode to run inside the IDE.

- [Hitechies](https://www.hitechies.com/github-copilot-metered-billing-june-2026/) GitHub Copilot completed its shift to usage-based billing on June 1. The $10 Pro plan now includes 1,500 monthly credits with Opus models removed; a new $100/month Copilot Max tier provides 20,000 credits for heavier workloads. Student premium model access ended on the same date.

## Open Source

- [MiniMax Blog](https://www.minimax.io/blog/minimax-m3) MiniMax released M3 on June 1, an open-weight model combining a 1-million-token context window, native multimodal computer use, and a new sparse attention architecture. It leads the open-weight SWE-Bench Pro leaderboard at 59.0%. Weights are available; training code and inference operators have not been released.

## Security

- [CISA](https://www.cisa.gov/known-exploited-vulnerabilities-catalog) CISA added CVE-2026-41091 and CVE-2026-45498 — both affecting Microsoft Defender — to the Known Exploited Vulnerabilities catalog, with a federal patch deadline of June 3. Fixes are available in Defender Antimalware Platform versions 1.1.26040.8 and 4.18.26040.7.

## Local

- [City of Toronto](https://www.toronto.ca/news/city-of-toronto-and-province-of-ontario-announce-update-for-gardiner-expressway-acceleration-and-reallocation-of-funds-as-part-of-the-ontario-toronto-new-deal/) Toronto and Ontario are finalizing the transfer of the Gardiner Expressway and Don Valley Parkway to the province under the Ontario-Toronto New Deal.
