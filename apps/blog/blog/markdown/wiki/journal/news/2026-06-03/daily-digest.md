## AI Industry

- [NBC News](https://www.nbcnews.com/business/corporations/anthropic-files-ipo-openai-rcna347897) Anthropic filed a confidential S-1 with the SEC on June 1, setting up one of the largest AI listings in history. The filing follows a $65 billion Series H that put the valuation near $965 billion, with a May 2026 revenue run-rate of approximately $47 billion.

- [Bloomberg](https://www.bloomberg.com/news/articles/2026-06-01/anthropic-s-first-mover-ipo-edge-set-to-widen-lead-over-openai) Anthropic's IPO filing landed ahead of OpenAI, which is expected to follow in the coming weeks. Both companies had previously targeted a fall debut.

- [CNBC](https://www.cnbc.com/2026/06/02/nvidias-new-pc-chips-are-ceos-bid-to-own-every-part-of-ai-stack.html) NVIDIA unveiled the RTX Spark superchip at Computex 2026, its first processor built for Windows PCs. The chip pairs a Blackwell RTX GPU with a 20-core Grace CPU on a TSMC 3nm package; systems arrive fall 2026.

- [NVIDIA Newsroom](https://nvidianews.nvidia.com/news/meta-builds-ai-infrastructure-with-nvidia) NVIDIA and Meta announced a multiyear strategic partnership covering on-premises, cloud, and AI infrastructure. Meta is deploying millions of Blackwell and Rubin GPUs and has committed $115–135 billion in AI capital expenditure for 2026.

- [Tom's Guide](https://www.tomsguide.com/news/live/google-io-2026-live-news-updates) Google I/O 2026 announcements included Gemini Spark, a 24/7 agentic assistant with Gmail and Workspace integration rolling out to AI Ultra subscribers. The Ultra subscription price dropped from $250 to $200 per month, with a new $100 Developer tier added.

## AI Tooling

- [Anthropic](https://www.anthropic.com/news) Claude Opus 4.8 launched with dynamic workflows in Claude Code — users can instruct Claude to orchestrate work across tens to hundreds of sub-agents on large-scale tasks. The release also adds effort control on claude.ai and a fast mode that is now three times cheaper than its predecessor.

- [Anthropic](https://techcrunch.com/2026/06/02/anthropic-scales-claude-mythos-to-critical-infrastructure-in-15-countries/) Anthropic expanded Project Glasswing to approximately 150 new organizations across 15+ countries, extending Claude Security to critical infrastructure operators. Claude Security scans codebases and suggests patches using Opus 4.8.

- [Thurrott.com](https://www.thurrott.com/dev/336962/build-2026-microsoft-announces-github-copilot-app) Microsoft announced the GitHub Copilot app at Build 2026 — a standalone agent-native desktop client for Windows, Mac, and Linux, available now in technical preview with a Copilot subscription.

- [Windows News](https://chatforest.com/builders-log/microsoft-build-2026-recap-windows-agent-platform-project-polaris-copilot-workspace/) Microsoft revealed Project Polaris, an in-house AI coding model that will replace GPT-4 Turbo as the default engine for GitHub Copilot starting August 2026. A three-month fallback period will be available for teams that want to stay on GPT-4.

- [HiTechies](https://www.hitechies.com/github-copilot-metered-billing-june-2026/) GitHub Copilot switched to token-based AI Credits billing on June 1, moving from a flat subscription to an open-ended flex model with $0.01-per-credit overages.

- [OpenTools](https://opentools.ai/news/microsoft-cancels-claude-code-licenses-copilot-cli) Microsoft's Experiences + Devices division told engineers to stop using Claude Code by June 30 and transition to GitHub Copilot CLI. The move affects the teams behind Windows, Microsoft 365, Outlook, Teams, and Surface.

- [DevToolPicks](https://devtoolpicks.com/blog/anthropic-splits-claude-subscriptions-agent-sdk-credit-june-2026) Starting June 15, programmatic Claude API usage via subscription plans shifts to a separate monthly credit pool, splitting personal and developer access.

## Security

- [The Cyber Express](https://thecyberexpress.com/cve-2025-48595-android-june-2026/) Google's June 2026 Android security update patches 124 vulnerabilities, including one actively exploited zero-day — CVE-2025-48595, an integer overflow in the Android Framework allowing local privilege escalation without user interaction. Affects Android 14, 15, and 16.

- [The Hacker News](https://thehackernews.com/2026/05/microsoft-warns-of-two-actively.html) CISA added two Microsoft Defender CVEs — CVE-2026-41091 and CVE-2026-45498 — to its Known Exploited Vulnerabilities catalog, with a remediation deadline of June 3, 2026 for federal agencies. CVE-2026-41091 can grant SYSTEM privileges when exploited.

- [Threat-Modeling.com](https://threat-modeling.com/vulnerability-intelligence-report-june-2-2026/) Palo Alto Networks PAN-OS and Prisma Access authentication bypass CVE-2026-0257 (CVSS 7.8) is under active exploitation, allowing attackers to establish unauthorized VPN connections.
