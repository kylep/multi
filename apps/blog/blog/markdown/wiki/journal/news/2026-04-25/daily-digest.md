## AI Industry

- [NVIDIA Newsroom](https://nvidianews.nvidia.com/news/meta-builds-ai-infrastructure-with-nvidia) NVIDIA and Meta announced a multiyear, multigenerational strategic partnership covering on-premises and cloud AI infrastructure. ## AI Tooling

- [Microsoft 365 Blog](https://www.microsoft.com/en-us/microsoft-365/blog/2026/04/22/copilots-agentic-capabilities-in-word-excel-and-powerpoint-are-generally-available/) Microsoft made Copilot Agent Mode generally available in Word, Excel, and PowerPoint on April 22, allowing users to delegate multi-step autonomous tasks within documents and spreadsheets. The change follows months of corporate pushback against forced AI integration.

- [Google Workspace Blog](https://workspace.google.com/blog/product-announcements/10-more-announcements-workspace-at-next-2026) Google launched a Workspace MCP Server in developer preview at Cloud Next 2026, letting developers connect Gmail, Drive, Calendar, and Chat to external AI agents and applications via the Model Context Protocol. Google also announced a data import tool in the admin console that speeds Microsoft 365-to-Workspace migrations by up to 5x.

## AI Industry

- [Bloomberg](https://www.bloomberg.com/news/articles/2026-04-24/google-plans-to-invest-up-to-40-billion-in-anthropic) Google is committing up to $40 billion to Anthropic — $10 billion now at a $350 billion valuation, with up to $30 billion more tied to performance milestones. The deal includes a five-year, five-gigawatt compute commitment from Google Cloud. Anthropic's annualized revenue crossed $30 billion this month.

- [TechCrunch](https://techcrunch.com/2026/04/23/openai-chatgpt-gpt-5-5-ai-model-superapp/) OpenAI released GPT-5.5 on April 23, the first fully retrained base model since GPT-4.5. It ships with a 1M-token context window and is priced at $5/$30 per million tokens in the API. ChatGPT now reports 900 million weekly active users and over 50 million subscribers.

- [Anthropic](https://www.anthropic.com/news/claude-opus-4-7) Claude Opus 4.7 launched on April 16 as Anthropic's most capable generally available model. It introduces task budgets — a public beta feature letting developers set rough token targets for full agentic loops — and posts strong scores on long-horizon coding and vision benchmarks.

## Open Source

- [Kubernetes Blog](https://kubernetes.io/blog/2026/04/22/kubernetes-v1-36-release/) Kubernetes v1.36 "Haru" shipped April 22. SELinux volume mounting is now generally available, replacing recursive file relabeling with a mount-time context option that cuts pod startup delays on SELinux systems. The long-running gogoprotobuf removal is complete.

## Security

- [The Hacker News](https://thehackernews.com/2026/04/hackers-exploit-cve-2025-55182-to.html) An active campaign exploiting CVE-2025-55182 (CVSS 10.0 RCE in Next.js React Server Components) has breached 766 hosts. Attackers deployed the NEXUS Listener framework to harvest AWS credentials, SSH keys, GitHub tokens, Stripe API keys, and database connection strings. The campaign is attributed to CL-STA-1015, an initial access broker with suspected ties to China's Ministry of State Security.

- [Cybersecurity News](https://cybersecuritynews.com/crowdstrike-logscale-vulnerability/) CVE-2026-40050 is a critical unauthenticated path-traversal flaw (CVSS 9.8) in CrowdStrike LogScale self-hosted versions 1.224.0–1.234.0 that allows arbitrary file reads without credentials. No active exploitation has been observed. SaaS clusters were network-blocked on April 7; self-hosted users should upgrade to 1.235.1 or 1.234.1.
