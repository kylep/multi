## AI Industry

- [NVIDIA Newsroom](https://nvidianews.nvidia.com/news/meta-builds-ai-infrastructure-with-nvidia) NVIDIA and Meta announced a multiyear, multigenerational strategic partnership covering on-premises and cloud AI infrastructure. Meta will deploy millions of NVIDIA Blackwell and Rubin GPUs across hyperscale data centers, scale out AI workloads with NVIDIA Spectrum-X Ethernet, and expand NVIDIA CPU deployment. NVIDIA Confidential Computing is included to enable AI capabilities while protecting user privacy.

## AI Tooling

- [Microsoft 365 Blog](https://www.microsoft.com/en-us/microsoft-365/blog/2026/04/22/copilots-agentic-capabilities-in-word-excel-and-powerpoint-are-generally-available/) Microsoft made Copilot Agent Mode generally available in Word, Excel, and PowerPoint on April 22, allowing users to delegate multi-step autonomous tasks within documents and spreadsheets. Expansion to Outlook, Teams, and OneNote is planned later in 2026.

- [BleepingComputer](https://www.bleepingcomputer.com/news/microsoft/microsoft-now-lets-admins-uninstall-copilot-on-enterprise-devices/) Microsoft now allows IT administrators to fully uninstall Copilot from enterprise devices via a new policy setting, available broadly after the April 2026 Patch Tuesday. The change follows months of corporate pushback against forced AI integration.

- [Google Workspace Blog](https://workspace.google.com/blog/product-announcements/10-more-announcements-workspace-at-next-2026) Google launched a Workspace MCP Server in developer preview at Cloud Next 2026, letting developers connect Gmail, Drive, Calendar, and Chat to external AI agents and applications via the Model Context Protocol. Google also announced a data import tool in the admin console that speeds Microsoft 365-to-Workspace migrations by up to 5x.

## Security

- [The Hacker News](https://thehackernews.com/2026/04/fortinet-patches-actively-exploited-cve.html) CVE-2026-35616 (CVSS 9.1) in Fortinet FortiClient EMS is a pre-authentication API access bypass that enables privilege escalation. Active exploitation attempts were first recorded March 31; Fortinet has released a patch.
