## AI Industry

- [CNBC](https://www.cnbc.com/2026/04/20/amazon-invest-up-to-25-billion-in-anthropic-part-of-ai-infrastructure.html) Amazon will invest up to $25 billion in Anthropic — $5 billion now at a $380B valuation, up to $20 billion more tied to commercial milestones — as part of a deal committing Anthropic to spend over $100 billion on AWS over the next decade. Anthropic expects to bring nearly 1 gigawatt of Trainium2 and Trainium3 capacity online by year-end, with a path to 5 gigawatts total.

- [Anthropic](https://www.anthropic.com/news/claude-opus-4-7) Anthropic released Claude Opus 4.7 on April 16 with baked-in cybersecurity guardrails — the model automatically detects and blocks requests indicating prohibited exploit or attack activity. Anthropic also launched a Cyber Verification Program for legitimate security researchers who need full access for penetration testing and red-teaming. Pricing holds at $5/million input tokens and $25/million output tokens.

- [TechCrunch](https://techcrunch.com/2026/04/20/google-rolls-out-gemini-in-chrome-in-seven-new-countries/) Google rolled out its Gemini AI assistant in Chrome to seven additional countries: Australia, Indonesia, Japan, the Philippines, Singapore, South Korea, and Vietnam. The desktop rollout covers all seven; the iOS rollout excludes Japan.

- [The Defense Post](https://thedefensepost.com/2026/04/20/pentagon-gemini-ai-google/) Google is in active talks with the US Department of Defense to deploy Gemini in classified environments, marking a renewed push into defense after the Project Maven controversy.

- [Microsoft Copilot Blog](https://www.microsoft.com/en-us/microsoft-copilot/blog/) Microsoft CEO Satya Nadella ordered a high-priority overhaul of Copilot on April 18, shifting to selling it as a standalone product at roughly $30 per user per month rather than bundled for free in Microsoft 365. An always-active version that monitors Outlook and calendars for proactive task suggestions is under development.

## AI Tooling

- [MacRumors](https://www.macrumors.com/2026/04/15/anthropic-rebuilds-claude-code-desktop-app/) Anthropic rebuilt the Claude Code desktop app around parallel sessions, adding a sidebar for managing multiple active sessions, drag-and-drop workspace layout, an integrated terminal, an in-app file editor, a rebuilt diff viewer, and an expanded preview pane supporting HTML, PDFs, and local servers. The April updates also introduced Routines — scheduled automations that bundle a prompt, a repo, and connectors to run on a schedule or trigger from GitHub events or API calls.

- [InfoQ](https://www.infoq.com/news/2026/04/cloudflare-dynamic-workers-beta/) Cloudflare launched Dynamic Workers into open beta as part of Agents Week 2026 (April 13–17), offering V8 isolate-based sandboxing for AI-generated code execution that starts in milliseconds and claims 100x better speed and memory efficiency versus containers. The week also shipped Sandboxes GA, Cloudflare Mesh, AI Gateway across 14+ providers, and Agent Memory. Cloudflare Pages is now in maintenance mode; Workers receives all new investment.

## Open Source

- [GitHub](https://github.com/google/adk-python) Google's Agent Development Kit (adk-python) has crossed 8,200 GitHub stars, providing a framework for building and deploying multi-agent systems with first-class support for Gemini models and tool orchestration.

- [GitHub](https://github.com/meta-llama/llama-stack) Meta's llama-stack surpassed 6,400 stars as a unified deployment stack for the Llama 4 model family, offering a standardized interface for inference, safety checks, and tool use across local and cloud environments.

## Security

- [Cybersecurity News](https://cybersecuritynews.com/microsoft-patch-tuesday-april-2026/) Microsoft's April 2026 Patch Tuesday addressed 168 vulnerabilities, including CVE-2026-32201, a SharePoint Server spoofing flaw actively exploited in the wild. A second zero-day (CVE-2026-33825) in Windows Defender was publicly disclosed with a working proof-of-concept enabling local privilege escalation to SYSTEM on fully patched Windows 10 and 11.

- [The Hacker News](https://thehackernews.com/2026/04/fortinet-patches-actively-exploited-cve.html) CISA added Apache ActiveMQ Classic CVE-2026-34197 (CVSS 8.8) to its Known Exploited Vulnerabilities catalog after confirming active exploitation in the wild. Federal agencies are required to apply patches by April 30. The flaw allows unauthenticated remote code execution.

The EU intends to approve a €90 billion loan for Ukraine and a 20th package of anti-Russia sanctions next week.
