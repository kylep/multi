## AI Industry

- [Google AI for Developers](https://ai.google.dev/gemini-api/docs/changelog) Google released Gemini 3.5 Flash to general availability, positioning it as its most capable model for sustained agentic and coding tasks. It outperforms Gemini 3.1 Pro on key benchmarks and costs less than half of comparable flagship models.

- [Google Cloud Blog](https://cloud.google.com/blog/products/ai-machine-learning/innovations-from-google-io-26-on-google-cloud) Google launched the Managed Agents API for Gemini in public preview, letting developers deploy stateful autonomous agents that run in secure, isolated, Google-hosted Linux sandbox environments. A general-purpose Antigravity Agent is also available in public preview.

- [Tom's Guide](https://www.tomsguide.com/news/live/microsoft-build-2026) Microsoft unveiled seven new MAI models at Build 2026, led by MAI-Thinking-1, a 35-billion-parameter reasoning model with a 256K context window built without distillation. MAI-Code-1-Flash, tuned for fast low-cost coding, began rolling out in VS Code via the GitHub Copilot model picker.

- [NVIDIA Newsroom](https://nvidianews.nvidia.com/news/meta-builds-ai-infrastructure-with-nvidia) NVIDIA confirmed Vera CPUs for data centers are in full production, with Anthropic, OpenAI, and SpaceXAI named as early customers. Jensen Huang described Vera as NVIDIA's new major growth driver alongside the boom in AI agents.

## AI Tooling

- [Engadget](https://www.engadget.com/2185601/microsoft-build-2026-live-blog-copilot-windows-news/) The GitHub Copilot native desktop app launched in preview at Microsoft Build 2026, bringing agentic workflows to a standalone application outside of any IDE or browser.

- [Microsoft Learn](https://learn.microsoft.com/en-us/partner-center/announcements/2026-june) Microsoft 365 Business Standard with Copilot and Microsoft 365 Business Premium with Copilot become permanent SKUs on July 1, 2026, at list prices of $23.50 and $32 per user per month. Copilot moves from add-on to core offering for business subscribers.

- [Cloudflare Blog](https://blog.cloudflare.com/whats-next-for-cloudflare-media/) Cloudflare AI Gateway now ships dollar-denominated spend limits, identity-driven budgets, per-user attribution, and PII redaction across all model providers in a single log line.

## Open Source

- [Google Blog](https://blog.google/innovation-and-ai/technology/ai/io-2026-google-ai/) Google open-sourced Gemini CLI, a terminal AI agent that brings the Gemini multimodal model into developers' command lines via a simple npx command. It supports code assistance, natural-language queries, Google Cloud integration, and can be embedded in CI/CD pipelines.

- [GitHub](https://github.com/lfnovo/open-notebook) open-notebook, an open-source alternative to Google's NotebookLM, climbed to 28,741 total stars with 4,245 gained this week. The project offers broader model support and custom integrations not available in the closed-source original.

## Security

- [Tenable](https://www.tenable.com/blog/microsofts-june-2026-patch-tuesday-addresses-198-cves-cve-2026-49160-cve-2026-50507) Microsoft's June 2026 Patch Tuesday addressed 198 CVEs — the largest single release since the program began — including 32 critical and three zero-days disclosed before patches were available. Notable fixes include a Windows BitLocker bypass (CVE-2026-50507) and a CTFMON elevation-of-privilege flaw granting SYSTEM privileges (CVE-2026-45586).

- [The Hacker News](https://thehackernews.com/2026/06/veeam-backup-replication-rce-flaw-lets.html) A remote code execution flaw in Veeam Backup & Replication (CVE-2026-44963) lets authenticated domain users run arbitrary code on backup servers. Patch available; organizations using Veeam in domain-joined environments should prioritize this update.

## Geopolitics

- [10 Things Global News](https://www.10things.news/p/10-things-global-news-10th-june-2026) Iran's IRGC launched attacks on US military bases in Bahrain, Kuwait, and Jordan on June 10, hitting 21 reported targets. Initial US assessments indicate nearly all missiles and drones were intercepted. The exchange followed a US Apache helicopter downing near the Strait of Hormuz and represents the largest escalation since the April ceasefire.

- [Reuters](https://www.reuters.com/) Anti-immigration protests spread across Northern Ireland, England, Scotland, and Wales after a man believed to be from Sudan was charged with attempted murder following a knife attack in Belfast. Prime Minister Keir Starmer condemned the violence; police urged communities not to incite hate.
