## AI Industry

- [CNBC](https://www.cnbc.com/2026/06/01/anthropic-ipo-s1-prospectus.html) Anthropic filed a confidential S-1 draft with the SEC on June 1, targeting an IPO against a $965 billion valuation. The company's revenue run rate reached $47 billion for 2026, up from $10 billion in all of 2025.

- [Google Blog](https://blog.google/innovation-and-ai/sundar-pichai-io-2026/) Google I/O 2026 launched Gemini 3.5 Flash as the new production model alongside Managed Agents in public preview — stateful, sandboxed AI agents running in Google-hosted Linux environments via the Gemini API. Gemini Spark, a personal AI agent, and a redesigned interface with a "Neural Expressive" design language were also announced.

- [Microsoft Blog](https://blogs.microsoft.com/blog/2026/06/02/microsoft-build-2026-be-yourself-at-work/) Microsoft Build 2026 (June 2) debuted MAI-Thinking-1, a 35-billion-parameter MoE reasoning model with a 256K context window, built on commercially licensed data without distillation from third-party models including OpenAI. It scores 94.5% on AIME 2026 and is available in private preview via Microsoft Foundry.

- [NVIDIA Newsroom](https://nvidianews.nvidia.com/news/meta-builds-ai-infrastructure-with-nvidia) Meta and NVIDIA announced a multiyear strategic infrastructure partnership. Meta will deploy millions of Blackwell and Rubin GPUs across hyperscale data centers, expand NVIDIA CPU use, and adopt Spectrum-X Ethernet and NVIDIA Confidential Computing.

## AI Tooling

- [Anthropic](https://releasebot.io/updates/anthropic) The Claude Agent SDK (v0.2.x, June 5) is now on PyPI for Python. Self-hosted sandboxes (public beta) move tool execution to operator-configured environments while the agent loop stays on Anthropic infrastructure. MCP tunnels (research preview) let agents reach MCP servers in private networks. Starting June 15, Agent SDK calls move to a separate credit pool — $20/mo for Pro, $100 for Max 5x, $200 for Max 20x.

- [Microsoft Build](https://microsoft.ai/news/microsoft-build-2026-mai-keynote-transcript/) MAI-Code-1-Flash began rolling out in VS Code through the GitHub Copilot model picker, tuned for fast, low-cost coding. Microsoft announced a Copilot Super App — combining GitHub Copilot, Copilot chat, and an agentic Autopilot capability — targeting a summer 2026 launch.

- [WinBuzzer](https://winbuzzer.com/2026/05/15/microsoft-starts-canceling-claude-code-licenses-xcxwbn/) Microsoft's Experiences & Devices division (Windows, Microsoft 365, Teams, Surface) began canceling internal Claude Code licenses in May, requiring engineers to move to GitHub Copilot CLI by June 30.

## Security

- [The Cyber Express](https://thecyberexpress.com/cve-2025-48595-android-june-2026/) Google's June 2026 Android security update patches 124 vulnerabilities including one actively exploited zero-day — CVE-2025-48595, an integer overflow in the Android Framework that allows local privilege escalation without user interaction.

- [Help Net Security](https://www.helpnetsecurity.com/2026/06/01/windows-netlogon-rce-exploited-cve-2026-41089/) CVE-2026-41089, a zero-click stack buffer overflow in Windows Netlogon (CVSS 9.8), is actively exploited against domain controllers on Windows Server 2012–2025. Belgium's Centre for Cybersecurity issued an emergency-patch advisory; Microsoft originally rated the bug "Exploitation Less Likely" when it disclosed it on May 12.
