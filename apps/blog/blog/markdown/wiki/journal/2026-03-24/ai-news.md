## AI Tooling

- [TechCrunch](https://techcrunch.com/2026/03/24/anthropic-hands-claude-code-more-control-but-keeps-it-on-a-leash/) Anthropic launched Auto Mode for Claude Code in research preview. The system reviews each agent action before executing — safe actions proceed automatically, risky ones and suspected prompt injections are blocked.

- [GeekWire](https://www.geekwire.com/2026/microsoft-launches-copilot-checkout-joining-the-ai-shopping-race-against-amazon-google-and-openai/) Microsoft launched Copilot Checkout, letting users complete purchases inside Copilot on Copilot.com via PayPal, Shopify, or Stripe. Etsy sellers launch first; Shopify merchants are auto-enrolled with an opt-out window.

- [Blockchain News](https://blockchain.news/news/google-expands-gemini-ai-google-tv-three-new-features) Google expanded Gemini on Google TV with three features: context-aware visual responses, educational deep-dive content, and live sports briefs with scorecard overlays. Rolling out in the US and Canada.

## Security

- [The Hacker News](https://thehackernews.com/2026/03/ubuntu-cve-2026-3888-bug-lets-attackers.html) CVE-2026-3888 (CVSS 7.8): a timing exploit in Ubuntu 24.04 and later allows a local unprivileged attacker to escalate to root via an interaction between snap-confine and systemd-tmpfiles. Affects default desktop installs; patch is available.

- [CISA](https://www.cisa.gov/known-exploited-vulnerabilities-catalog) CVE-2026-22719 (CVSS 8.1): unauthenticated command injection in VMware Aria Operations added to CISA's Known Exploited Vulnerabilities catalog. Federal agencies had a March 24 remediation deadline; active exploitation confirmed in the wild.
