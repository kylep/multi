# Blog Distribution

Plans and implementation notes for syndicating blog posts from
kyle.pericak.com to external platforms.

## Channels

- [X/Twitter Auto-Post (PER-26)](x-twitter-auto-post.md) — planned
- Dev.to RSS Syndication (PER-24) — backlog
- Hashnode Cross-Post (PER-25) — backlog
- Substack Cross-Post (PER-23) — backlog
- Reddit / Subreddit (PER-28) — backlog

## Principles

- kyle.pericak.com is the source of truth. All cross-posts set
  canonical URL back to the original.
- Automate where possible. Manual cross-posting doesn't scale.
- RSS feed at kyle.pericak.com/feed.xml is the syndication trigger.
