# Background Pattern (Tileable)

**Filename:** `background-pattern.png`  
**Size:** 400×400  
**Format:** PNG, no transparency (solid background)

## Prompt

A subtle, tileable background pattern for a whimsical mobile app. Very light blue (#EBF4FF) base color. Scattered across the tile are tiny, faint doodle elements at about 8-10% opacity: miniature stars, swirls, dots, tiny smiley faces, small hearts, and confetti shapes. All elements in slightly darker blue (#ADD9FF). The pattern tiles seamlessly on all four edges. Minimal, not busy — should be barely noticeable behind foreground content. Clean, modern, hand-drawn doodle aesthetic. 400x400 pixels, seamlessly tileable.
