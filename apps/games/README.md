# Games
My son Oliver thinks he might want to be a game developer when he grows up.
I made this section to put little games we work on together in to help him learn.

The actively maintained game is [robot-battle](robot-battle/). Older
experiments (snake, robotext-battle, sillyapp, kid-bot-battle-sim) moved
to [kylep/multi-sandbox](https://github.com/kylep/multi-sandbox).
