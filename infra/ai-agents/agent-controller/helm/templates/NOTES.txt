Agent Controller deployed to namespace {{ .Values.namespace }}.

1. Check controller status:
   kubectl get pods -n {{ .Values.namespace }}

2. View AgentTasks:
   kubectl get agenttasks -n {{ .Values.namespace }}

3. Create a manual task:
   kubectl apply -f config/samples/manual-analyst.yaml

4. Trigger via webhook:
   kubectl port-forward -n {{ .Values.namespace }} svc/{{ include "agent-controller.fullname" . }} 8080:8080
   curl -X POST http://localhost:8080/webhook \
     -H "Authorization: Bearer $AI_WEBHOOK_TOKEN" \
     -H "Content-Type: application/json" \
     -d '{"agent":"analyst","prompt":"list wiki pages"}'

5. View agent job logs:
   kubectl logs -n {{ .Values.namespace }} -l app.kubernetes.io/managed-by=agent-controller
